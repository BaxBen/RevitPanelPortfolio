# MainRevitPanel

Плагин для Autodesk Revit (Мини-портфолио). Содержит 2 инструмента для анализа инженерных сетей и 2 инструмента для экспорта проекта в DWG и EXCEL: 1. Проверяет пересечение между трубами и вентиляцией 2. Ищет путь до кранов в системе 3. Выводит спецификации в EXCEL 4. Экспортирует листы в формат DWG 5. Экспортирует модель в IFC + Интеграция с GIP VISION


## Описание задач

Плагин реализует 5 независимые команды, вызываемые из панели Revit:

### 1. Поиск пересечений между инженерными системами.
Проверяет пересечения между воздуховодами и трубопроводами, создает отчет и подсвечивает проблемные места.

**Особенности:**
- Геометрический анализ, Solid операции
- Работа с Json
- Формирование отчета в Json

### 2. Поиск ближайшего и дальнего крана. (В доработке.)
Пользователь выделяет один экземпляр семейства "Шаровый кран с муфтой". Плагин строит граф соединений и находит в системе все краны того же типа.

**Вывод информации** (WPF-окно):
- Имя крана
- ID крана
- ID всех коннекторов и труб участвующих в пути
- Длина участка

### 3. Экспорт спецификаций в Excel
Экспортирует спецификации в файлы с расширением Excel.

**Особенности:**
- Работа с библиотекой EPPlus

### 4.Экспорт листов в DWG (Можно улучшить.)
Экспортирует листы в файлы с расширением DWG.

### 5.Экспорт модели в IFC (Можно улучшить. В будущем будет сохранять ключ)
Экспортирует модель в IFC + Интеграция с GIP VISION
**Особенности:**
- Работа с API GIP VISION
- RestAPI

## Требования

- **Revit**: 2022
- **VS**: 2022-2025


## Установка
1. Скачайте последний релиз
2. В свойстве проекта прописан путь для быстрого копирования исполняемых файлов, если Свойства -> События сборки пусто, то вставте 
- xcopy "$(TargetDir)*.*" "C:\Program Files\RevitPanel\" /y /i
- xcopy "$(ProjectDir)$(ProjectName).addin" "%appdata%\Autodesk\Revit\Addins\2022\" /y /i
- RevitPanel - Место где будет хранится все файлы .dll
3. Запустите Revit

## Если отсутствует VS
1. Скачайте последний релиз
2. Используйте SetupPanel.msi
3. Не меняйте путь установки
4. Запустите Revit
