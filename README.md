# MainRevitPanel

Плагин для Autodesk Revit (Мини-портфолио). Содержит 2 инструмента для анализа инженерных сетей и 2 инструмента для экспорта проекта в DWG и EXCEL: 1. Проверяет пересечение между трубами и вентиляцией, 2. Ищет ближайший и самый дальний однотипные элементы в системе, 3. Выводит спецификации в EXCEL, 4. Экспортирует листы в формат DWG


## Описание задач

Плагин реализует 4 независимые команды, вызываемые из панели Revit:

### 1. Поиск пересечений между инженерными системами.
Проверяет пересечения между воздуховодами и трубопроводами, создает отчет и подсвечивает проблемные места.

**Особенности:**
- Геометрический анализ, Solid операции
- Работа с Json
- Формирование отчета в Json

### 2. Поиск ближайшего и дальнего крана. (В доработке.)
Пользователь выделяет один экземпляр семейства "Шаровый кран с муфтой". Плагин строит граф соединений и находит в системе два крана того же типа:

- **Ближайший** — минимальное количество соединений от исходного
- **Самый дальний** — максимальное количество соединений (длина пути)

**Вывод информации** (WPF-окно):
- ID исходного крана
- ID ближайшего крана и ID труб и коннектеров до него
- ID самого дальнего крана и ID труб и коннектеров до него

### 3. Экспорт спецификаций в Excel
Экспортирует спецификации в файлы с расширением Excel.

**Особенности:**
- Работа с библиотекой EPPlus

### 4.Экспорт листов в DWG (Можно улучшить.)
Экспортирует листы в файлы с расширением DWG.


## Требования

- **Revit**: 2022
- **VS**: 2022-2025


## Установка
1. Скачайте последний релиз
2. В свойстве проекта прописан путь для быстрого копирования исполняемых файлов, если Свойства -> События сборки пусто, то вставте
xcopy "$(TargetDir)*.*" "C:\Program Files\RevitPanel\" /y /i
xcopy "$(ProjectDir)$(ProjectName).addin" "%appdata%\Autodesk\Revit\Addins\2022\" /y /i
RevitPanel - Место где будет хранится все файлы .dll
3. Запустите Revit

## Если отсутствует VS
1. Скопируйте папку RevitPanel 
2. Разместите по пути C:\Program Files\
3. Поместите .addin файл в %APPDATA%\Autodesk\Revit\Addins\2022
4. Запустите Revit
